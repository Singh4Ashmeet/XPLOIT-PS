XPLOIT Vault System — Automated Solver

A fully automated Python solve script for the XPLOIT Vault System challenge.
The script patches the challenge binary, recreates the required vault state, computes the unlock key dynamically, and unlocks the vault automatically.

This project demonstrates binary patching, reverse engineering logic reconstruction, and automated exploit scripting.

Features

Binary patching using direct byte manipulation

Automatic creation of .vault_state

Dynamic computation of vault unlock key

Automatic interaction with the patched binary

Fully automated solve process

How It Works

The solve script performs three major steps.

1. Binary Patching

The script modifies the challenge binary at specific offsets.

Patch 1

Changes authentication level from 1 to 999.

Original:
movl $0x1, -0x54(%rbp)

Patched:
movl $0x3e7, -0x54(%rbp)

This bypasses the low privilege authentication check.

Patch 2

Redirects execution flow after authentication.

Original:
call puts("Terminating session...")

Patched:
call unlock_vault_sequence

This forces the program to directly trigger the vault unlock sequence.

2. Vault State Initialization

On the first execution, the program generates a hidden file:

.vault_state

The first byte of this file becomes part of the unlock key calculation.

3. Unlock Key Calculation

The vault unlock logic reconstructs the key as:

key = strlen(argv[0]) XOR pid_seed XOR vault_byte

Where:

vault_byte → first byte from .vault_state

pid_seed → (pid >> 8) XOR (pid & 0xFF)

strlen(argv0) → length of program name

The script calculates this value automatically and sends it to the binary.

Usage

Place the challenge binary in the same directory and run:

python3 solve.py

The script will:

Patch the binary

Create .vault_state

Compute the unlock code

Automatically unlock the vault
